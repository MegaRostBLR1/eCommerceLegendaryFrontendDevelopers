- **Что было сделано:** Создал тест-кейсы
**api-client:**
- GET-запрос без токена → проверяю, что `Authorization` не добавляется
- Наличие токена в `localStorage` → проверяю добавление заголовка
- Ошибка 404 → проверяю выброс `ApiError` с правильным `status`
- POST с телом → проверяю передачу `body` и `method`
- Слияние заголовков → кастомные + дефолтные не конфликтуют

**authorization-service:**
- Декодирование токена → мокаю `jwtDecode`, проверяю возврат данных
- Валидный токен → `isAuthUser()` возвращает `true`
- Просроченный токен → `isAuthUser()` возвращает `false`
- Определение админа → `userIsAdmin()` проверяет `role === 'admin'`
- Ошибки декодирования → `try/catch` возвращает `false`, не крашит приложение

**user-service:**
- Получение профиля → мок `apiClient.request`, проверка URL `/users/${id}`
- Обновление профиля → проверка `method: 'PATCH'` и `JSON.stringify(data)`
- Удаление сервиса → проверка `method: 'DELETE'`
